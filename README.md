# keystone-bot-js
> Tracks weekly keystones

## Usage

Set a keystone
```sh
-keybot set dungeon level [alt]
```
Unset a keystone
```sh
-keybot unset [alt]
```

Deplete a keystone
```sh
-keybot deplete [alt]
```

List all keystones
```sh
-keybot list
```
